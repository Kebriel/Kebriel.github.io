Capture the Flag 2.0

My main significant project at the moment. The result of several months of nonstop code revision and personal advancement, Capture the Flag 2.0 is a revitalized version of an older project with the same name -- but now thoroughly and totally rewritten from the bottom-up. This time around, my focus was on optimization of performance and memory usage, minimization of latency, code readability, as well as building innovative systems to expand behavior and overcome limitations.

Think of CTF 2.0 not merely as a minigame, but as a gallery to showcase numerous applied distinct theoretical concepts, each meant to tackle what I saw to be a limitation or inefficiency in an element of minigame building and function.
