Capture the Flag 2.0

My main project at the moment. The result of several months of nonstop code revision and personal advancement, Capture the Flag 2.0 is a revitalized version of my much older project with the same name -- now thoroughly and totally rewritten from the bottom-up. It is a very hefty & substantial project into which I poured hundreds of hours, so in many ways, it is numerous smaller projects contained within a project.

CTF 2.0 should be thought of not merely as a minigame, but as a gallery to showcase numerous applied distinct theoretical frameworks, each meant to expand behavior and/or simplify a process that otherwise might often be unintuitive and error-prone.

You can find the full project on GitHub, along with a lengthy README detailing most of CTF's especially noteworthy features. While CTF 2.0 is somewhat thoroughly notated, I would also recommend the README as a guide/ follow-along in the event that a certain stretch of the code doesn't quite make sense or bears further explanation.
